## :fire: How to

- ### **Requirements**

  - It's **necessary** **[Node.js](https://nodejs.org/en/)**

1. Clone the repo :

```sh
  > git clone https://github.com/Cancellier27/web-app-fc
```

2. Install dependencies:

```sh
  > npm install
```

3. Execute the app:

```sh
  > npm start
```
